# Refile::Watermark

A Refile image processor to enable creation of watermarks

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'refile-watermark'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install refile-watermark
